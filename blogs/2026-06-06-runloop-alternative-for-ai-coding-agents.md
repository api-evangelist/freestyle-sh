Runloop is a serious platform for AI coding agents with sandboxed execution environments, VM technology, Git integration, browser capabilities, and file operations, but as agent products shift from executing work for the agent to hosting the user's live development machine, Freestyle VMs offer superior workspace capabilities. Freestyle preserves both disk and in-memory state, maintains long-lived interactive terminal sessions, supports live forking at decision points, and treats the VM itself as the core product primitive.
