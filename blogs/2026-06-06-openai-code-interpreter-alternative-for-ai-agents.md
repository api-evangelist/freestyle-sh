OpenAI Code Interpreter excels at Python computation within conversations but falls short for agent products requiring persistent, complex environments where state should survive the product loop across terminal sessions, running services, logs, and file systems. Freestyle VMs provide durable, long-lived machines rather than ephemeral containers, enabling users and agents to access and resume their environments across multiple interactions.
