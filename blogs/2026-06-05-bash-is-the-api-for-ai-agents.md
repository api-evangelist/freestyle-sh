Bash keeps emerging in agent products because it represents the shortest path from intent to actual machine behavior, enabling agents to install packages, inspect logs, run migrations, and use specialized utilities that pre-designed tool schemas cannot anticipate. Narrow tool schemas age quickly and become product limitations whenever needed capabilities are missing, making shell access essential for agents that need to operate across unpredictable real-world tasks.
